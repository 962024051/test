# 静态方法

# fromCharCode()

通过一串 Unicode 创建字符串

String.fromCharCode(num1[, ...[, numN]])
参数 描述
numN UTF-16 代码
返回值 字符串

# fromCodePoint()

String.fromCodePoint(num1[, ...[, numN]])
参数 描述
numN 一串 Unicode 编码位置，即“代码点
返回值 指定的 Unicode 编码位置创建的字符串

# raw()

模板字符串的标签函数

String.raw(callSite, ...substitutions)

String.raw`templateString`
参数 描述
callSite 一个模板字符串的“调用点对象”。类似{ raw: ['foo', 'bar', 'baz'] }。
...substitutions 任意个可选的参数，表示任意个内插表达式对应的值
templateString 模板字符串，可包含占位符（${...}）
返回值 返回原始字符串

# 实例属性

属性 描述
constructor 构造器函数
length 字符串长度

# 实例方法

# JavaScript 字符

# charAt()

str.charAt(index)
参数 描述
index 字符串位置(0到字符串长度-1)
返回值 返回index位置字符。

# charCodeAt()

str.charCodeAt(index)
参数 描述
index 字符串位置(0到字符串长度-1)
返回值 返回index位置UTF-16代码。

# codePointAt()

str.codePointAt(pos)
参数 描述
pos 字符串中需要转码的元素的位置
返回值 返回pos位置Unicode 编码点值的非负整数

# normalize()

str.normalize([form])
参数 描述
form 四种 Unicode 正规形式（Unicode Normalization Form）
"NFC"、"NFD"、"NFKC"，或 "NFKD" 其中的一个, 默认值为 "NFC"。
返回值 返回form位置Unicode 规范化形式的字符串

# 操作字符方法

# 合并字符串

str.concat(str2, [, ...strN])
参数 描述
str2 [, ...strN] 其他字符串
返回值 新字符串=原字符串+多个字符串

# 重复字符串

str.repeat(count)
参数 描述
count 0-无穷大的整数
返回值 新字符串=原字符串+重复字符串

# 替换字符串

str.replace(regexp|substr, newSubStr|function)
str.replace(正则表达式|字符串, 新字符串|函数调用字符串)

分割字符串数组

str.split([separator[, limit]])
参数 描述
separator 分割的字符串
limit 整数，分割数组
返回值 分割的字符串数组

# 提取字符串

slice()

str.slice(beginIndex[, endIndex])
参数 描述
beginIndex 字符串开始索引位置，默认为0
endIndex(可选) 字符串结束索引位置，默认为0
返回值 新字符串=索引位置提取原字符串

substr()

提取字符串

str.substr(start[, length])
参数 描述
start 字符串开始索引位置，默认为0
length 提取字符数
返回值 新字符串=索引位置提取原字符串

substring()

提取字符串

str.substring(start[, length])
参数 描述
start 字符串开始索引位置，默认为0
length 提取字符数
返回值 新字符串=索引位置提取原字符串

# 比较字符串

referenceStr.localeCompare(compareString[, locales[, options]])
原始字符串.localeCompare(比较字符串，设置区域语言类型)
  • 当 引用字符串 在 比较字符串 前面时返回 -1
  • 当 引用字符串 在 比较字符串 后面时返回 1
  • 相同位置时返回 0

# 索引搜索方法

# 返回布尔类型

在原字符串搜索包含某个字符串，返回true 或 false。

开始位置搜索字符串
str.startsWith(searchString[, position])

结尾位置搜索字符串
str.endsWith(searchString[, length])

任意位置搜索字符串
str.includes(searchString[, position])
参数 描述
searchString 搜索的字符串
position 默认为0，字符串索引位置
length 默认值字符串长度，字符串索引位置

# 返回索引位置

在原字符串搜索包含某个字符串，返回索引位置或-1。

开始位置搜索字符串
str.indexOf(searchValue [, fromIndex])

结尾位置搜索字符串
str.lastIndexOf(searchValue[, fromIndex])
参数 描述
searchValue 搜索的字符串
fromIndex 某方向字符串位置开始搜索

# 正则表达式

方法 描述
str.match(regexp) 返回Array，其内容取决于global（g）标志的存在与否，如果未找到匹配则为null。
str.search(regexp) 成功返回索引，失败返回-1

# 复制填充字符串

右边复制填充
str.padEnd(targetLength [, padString])
左边复制填充
str.padStart(targetLength [, padString])
参数 描述
targetLength 复制长度(整数)
padString(可选) 复制填充字符串
返回值 原字符串+填充复制字符串=新字符串

# 大小写转换

指定区域语言类型 转换小写格式
str.toLocaleLowerCase()
str.toLocaleLowerCase(locale) 
str.toLocaleLowerCase([locale, locale, ...])

指定区域语言类型 转换大写格式
str.toLocaleUpperCase()
str.toLocaleUpperCase(locale) 
str.toLocaleUpperCase([locale, locale, ...])

转换小写字符串
str.toLowerCase()

转换大写字符串
str.toUpperCase()
