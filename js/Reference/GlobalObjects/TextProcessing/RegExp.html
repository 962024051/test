# RegExp

# 构造函数

new RegExp(pattern[, flags])
RegExp(pattern[, flags])
参数 描述
pattern 正则表达式文本
flags 标识符字符串(gimuy) 任意组合

# 静态属性

访问器属性返回RegExp 的构造器

RegExp[Symbol.species]

指定下一次匹配的起始索引

lastIndex = regExpObj.lastIndex;

# 实例属性

属性 返回值
flags 含有 RegExp 对象 flags 的字符串
dotAll dotAll的值是布尔对象，如果使用了 "s" 标志，则返回 true；否则返回 `false
global global的值是布尔对象，如果使用了 "g" 标志，则返回true；否则返回false
ignoreCase ignoreCase 的值是布尔对象，如果使用了"i" 标志，则返回 true；否则，返回 false。
multiline multiline是一个布尔对象，如果使用了 "m" 标志，则返回true；否则，返回false
source 正则表达式的字符串文本。
sticky sticky 的值是布尔对象，如果使用了 "y" 标志，则返回 true；否则返回 `false
unicode unicode 的值是 Boolean，并且如果使用了 "u" 标志则为 true；否则为 false

# 实例方法

# exec()

一个指定字符串中执行一个搜索匹配。返回一个结果数组或 null

regexObj.exec(str)
参数 描述
str 正则表达式的字符串
返回值 匹配成功，返回一个数组。
匹配失败，返回null

# test()

regexObj.test(str)
参数 描述
str 正则表达式的字符串
返回值 如果正则表达式与指定的字符串匹配 ，返回true；否则false

# [@@match]()

regexp[Symbol.match](str)
参数 描述
str 正则表达式的字符串
返回值 匹配成功，返回一个数组。
匹配失败，返回null

# [@@matchAll]()

regexp[Symbol.matchAll](str)
参数 描述
str 正则表达式的字符串
返回值 匹配成功，返回一个迭代器。
匹配失败，返回null

# [@@replace]()

regexp[Symbol.replace](str, newSubStr|function)
参数 描述
str 正则表达式的字符串
newSubStr 替换的字符串
function 替换的回调函数的字符串
返回值 用替换器替换相应匹配项后的新字符串

# [@@search]()

regexp[Symbol.search](str)
参数 描述
str 正则表达式的字符串
返回值 搜索成功，返回一个索引位置。
搜索失败，返回-1

# [@@split]()

regexp[Symbol.split](str[, limit])
参数 描述
str 切割操作的目标字符串
limit(可选) 限制切割数量的特定整数
返回值 成功，返回一个数组。
失败，返回-1
