# TypedArray

一个类型化数组对象

描述了一个底层的二进制数据缓冲区的一个类数组视图

# 类型对象

类型 单个元素值的范围 大小(bytes) 描述 Web IDL 类型 C 语言中的等价类型
Int8Array -128 to 127 1 8 位二进制有符号整数 byte int8_t
Uint8Array 0 to 255 1 8 位无符号整数（超出范围后从另一边界循环） octet uint8_t
Uint8ClampedArray 0 to 255 1 8 位无符号整数（超出范围后为边界值） octet uint8_t
Int16Array -32768 to 32767 2 16 位二进制有符号整数 short int16_t
Uint16Array 0 to 65535 2 16 位无符号整数 unsigned short uint16_t
Int32Array -2147483648 to 2147483647 4 32 位二进制有符号整数 long int32_t
Uint32Array 0 to 4294967295 4 32 位无符号整数 unsigned long uint32_t
Float32Array 1.2×10-38 to 3.4×1038 4 32 位 IEEE 浮点数（7 位有效数字，如 1.1234567） unrestricted float float
Float64Array 5.0×10-324 to 1.8×10308 8 64 位 IEEE 浮点数（16 有效数字，如 1.123...15) unrestricted double double
BigInt64Array -263 to 263-1 8 64 位二进制有符号整数 bigint int64_t (signed long long)
BigUint64Array 0 to 264-1 8 64 位无符号整数 bigint uint64_t (unsigned long long)

# from()

从类数组对象或者可迭代对象中创建一个新的数组实例

TypedArray.from(source[, mapFn[, thisArg]])
参数 描述
source 转换成数组的伪数组对象或可迭代对象
mapFn 新数组中的每个元素会执行该回调函数
thisArg 执行回调函数 mapFn 时 this 对象
返回值 返回一个新的数组实例

# copyWithin()

在数组内部，将一段元素序列拷贝到另一段元素序列上，覆盖原有的值

typedarray.copyWithin(target[, start[, end]])
参数 描述
target 整数，该数组索引位置开始计算浅复制，默认值0
start(可选) 开始索引位置开始拷贝,默认值为0
end(可选) 结束索引位置停止拷贝，默认值为 arr.length
返回值 返回修改后的原数组。
# fill()

用一个固定值填充一个数组中从起始索引到终止索引内的全部元素

typedarray.fill(value[, start[, end]])
参数 描述
value 填充元素值
start(可选) 开始索引位置，默认值0
end(可选) 结束索引位置，默认值arr.length
返回值 返回修改后的原数组。

# 排序方法

# sort()

数据元素排序

typedarray.sort([compareFunction])
参数 描述
compareFunction(可选) 指定排列的函数
compareFunction(a,b)
compareFunction(第一个比较元素，第二个比较元素)
返回值 返回排序后的原数组

  • compareFunction(a, b) 小于 0 ，a 会被排列到 b 之前；

  • compareFunction(a, b) 等于 0 ， a 和 b 的相对位置不变。；

  • compareFunction(a, b) 大于 0 ， b 会被排列到 a 之前。

  • compareFunction(a, b) 必须总是对相同的输入返回相同的比较结果，否则排序的结果将是不确定的。

# 搜索方法

# includes()

判断一个数组是否包含一个指定的值.返回bool类型

typedarray.includes(valueToFind[, fromIndex])
参数 描述
valueToFind 查找的元素值
fromIndex(可选) 按索引位置开始查找搜索,默认值0
返回值 返回一个布尔类型，成功true,失败false
# indexOf()

判断一个数组是否包含一个指定的值.返回数组索引位置，失败返回-1

typedarray.indexOf(searchElement[, fromIndex])
参数 描述
searchElement 查找的元素值
fromIndex(可选) 按索引位置开始查找搜索,默认值0
返回值 返回成功数组索引位置,返回失败-1
# lastIndexOf()

返回指定元素在数组中的最后一个的索引，如果不存在则返回 -1。从数组的后面向前查找

typedarray.lastIndexOf(searchElement[, fromIndex])
参数 描述
searchElement 查找的元素值
fromIndex(可选) 按逆向索引位置开始查找搜索,默认值(arr.length - 1)
返回值 返回成功数组中该元素最后一次出现的索引,返回失败-1

typedarray.迭代方法(callback(currentValue [, index [, array]])[, thisArg])
迭代方法参数 描述

# 位置方法

实例方法 返回
find() 数组中第一个满足所提供函数的元素的值，否则返回 undefined
findIndex() 数组中通过提供函数的第一个元素的索引。否则，返回-1

# 迭代方法

实例方法 返回
forEach() 返回undefined。
some() 传入的函数只要有一项返回 true，它就会返回 true。
every() 传入的函数必须对每一项都返回 true，它才会返回 true；否则，它就返回 false
filter() 传入的函数对每一项进行筛选，并把筛选的值放在一个新数组，并返回出去，
没有新数组返回空的数组
map() 传入的函数对每一项执行操作，并把执行后的值放在一个新数组，并返回出去，
没有新数组返回空的数组

# 归并方法

arr.归并迭代方法(callback(accumulator, currentValue[, index[, array]])[, initialValue])
归并迭代方法参数 描述
callback() 数组每个元素执行函数，函数有三个参数
initialValue(可选) 首次调用 callback 函数时，累加器 accumulator 的值。
如果未提供该初始值，则将使用数组中的最后一个元素，并跳过该元素。
callback()参数 描述
accumulator 累加器：上一次调用回调函数时，回调函数返回的值。
首次调用回调函数时，如果 initialValue 存在，累加器即为 initialValue，
否则须为数组中的最后一个元素
currentValue 数组中正在处理的当前元素
index(可选) 数组中正在处理的当前元素的索引
array(可选) 当前迭代方法的数组
归并实例方法 描述
reducer() 从左到右为每个数组元素执行一次回调函数，
并把上次回调函数的返回值放在一个暂存器中传给下次回调函数，
并返回最后一次回调函数的返回值
reduceRight() 从右到左为每个数组元素执行一次回调函数，
并把上次回调函数的返回值放在一个暂存器中传给下次回调函数，
并返回最后一次回调函数的返回值。
