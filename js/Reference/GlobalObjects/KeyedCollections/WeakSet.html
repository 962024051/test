# WeakSet

对象允许你将弱保持对象存储在一个集合中。

# 构造函数

new WeakSet([iterable])
参数 描述
iterable 一个数组或者其他 iterable 对象

# 实例属性

实例属性 描述
constructor 返回一个函数，它创建了实例的原型。默认是Set函数

# 实例方法

# 操作

# delete()

ws.delete(key);
参数 描述
key 元素的键。
返回值 元素存在并删除返回 true；元素不存在返回 false。

# add()

向一个 WeakSet 对象的末尾添加一个指定的值

ws.add(value);
参数 描述
value 元素的值。
返回值 Set 对象

# has()

根据 WeakSet 是否存在相应对象返回布尔值

ws.has(key);
参数 描述
key 元素的键。
返回值 元素存在返回 true；元素不存在返回 false。
