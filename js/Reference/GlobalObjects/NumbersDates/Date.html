# Date

日期时间

# 构造函数

new Date();

new Date(value);
参数 描述
value 时间戳
new Date(dateString);
参数 描述
dateString 日期字符串
new Date(year, monthIndex [, day [, hours [, minutes [, seconds [, milliseconds]]]]]);
参数 是否可选 描述
year 年份
monthIndex 月份
day 可选 天数
hours 可选 小时
minutes 可选 分钟
seconds 可选 秒
milliseconds 可选 毫秒

# 静态属性

# Date.prototype

属性特性 值
writable false
enumerable false
configurable true

# Date.length

Date.length 的值是 7。这是该构造函数可接受的参数个数。

# 静态方法

# Date.now()

返回自 1970 年 1 月 1 日 00:00:00 (UTC) 到当前时间的毫秒数

# Date.parse()

Date.parse(dateString)
等于
new Date(dateString).getTime()
参数数 描述
dateString 日期格式的字符串
返回值 返回从 1970-1-1 00:00:00 UTC 到该日期对象的毫秒数

# Date.UTC()

Date.UTC(year,month[,date[,hrs[,min[,sec[,ms]]]]])
参数数 描述
year 1900 年后的某一年份
month 0 到 11 之间的一个整数，表示月份
date 1 到 31 之间的一个整数，表示某月当中的第几天
hrs 0 到 23 之间的一个整数，表示小时。
min 0 到 59 之间的一个整数，表示分钟
sec 0 到 59 之间的一个整数，表示秒。
ms 0 到 999 之间的一个整数，表示毫秒
返回值 返回从 1970-1-1 00:00:00 UTC 到该日期对象的毫秒数

# 实例属性

Date.prototype.constructor

返回创建了实例的构造函数，默认是 Date 构造函数。

# 实例方法

# 获取实例方法

实例方法 返回值
getDate() 返回本地时间对象某月中的第几天(1-31)
getDay() 返回本地时间对对象某星期中的第几天(0-6)
getFullYear() 返回本地时间对象的年份(四位数年份)
getHours() 返回本地时间对对象的小时(0-23)
getMilliseconds() 返回本地时间对对象的毫秒(0-999)
getMinutes() 返回本地时间对对象的分钟(0-59)
getMonth() 返回本地时间对对象的月份(0-11)
getSeconds() 返回本地时间对对象的秒数(0-59)
getTime() 返回从 1970-1-1 00:00:00 UTC 到该日期经过的毫秒数
getTimezoneOffset() 返回当前时区的时区偏移 单位分钟。
getUTCDate() 返回世界时间对象某月中的第几天(1-31)
getUTCDay() 返回世界时间对对象某星期中的第几天(0-6)
getUTCFullYear() 返回世界时间对象的年份(四位数年份)
getUTCHours() 返回世界时间对对象的小时(0-23)
getUTCMilliseconds() 返回世界时间对对象的毫秒(0-999)
getUTCMinutes() 返回世界时间对对象的分钟(0-59)
getUTCMonth() 返回世界时间对对象的月份(0-11)
getUTCSeconds() 返回世界时间对对象的秒数(0-59)

# 设置实例方法

# 年份

dateObj.setFullYear(yearValue[, monthValue[, dayValue]])
dateObj.setUTCFullYear(yearValue[, monthValue[, dayValue]])
参数 描述
yearValue 年份（四位数）
monthValue 月份（0-11）
dayValue 天数（1-31）

# 月份

dateObj.setMonth(monthValue[, dayValue]) dateObj.setUTCMonth(monthValue[,
dayValue])
参数 描述
monthValue 月份（0-11）
dayValue 天数（1-31）

# 天

dateObj.setDate(dayValue);
dateObj.setUTCDate(dayValue);
参数 描述
dayValue 天数(1-31)

# 小时

dateObj.setHours(hoursValue[, minutesValue[, secondsValue[, msValue]]])
dateObj.setUTCHours(hoursValue[, minutesValue[, secondsValue[, msValue]]])
参数 描述
hoursValue 小时(0-23)
minutesValue 分钟(0-59)
secondsValue 秒数(0-59)
msValue 毫秒(0-999)

# 分钟

dateObj.setMinutes(minutesValue[, secondsValue[, msValue]])
dateObj.setUTCMinutes(minutesValue[, secondsValue[, msValue]])
参数 描述
minutesValue 分钟(0-59)
secondsValue 秒数(0-59)
msValue 毫秒(0-999)

# 秒

dateObj.setSeconds(secondsValue[, msValue])
dateObj.setUTCSeconds(secondsValue[, msValue])
参数 描述
secondsValue 秒数(0-59)
msValue 毫秒(0-999)

# 毫秒

dateObj.setMilliseconds(millisecondsValue)
dateObj.setUTCMilliseconds(millisecondsValue)
参数 描述
millisecondsValue 毫秒(0-999)

# 时间

dateObj.setTime(timeValue)
参数 描述
timeValue 一个整数，从 1970-1-1 00:00:00 UTC 开始计时的毫秒数
返回值 UTC 1970 年 1 月 1 日 00:00:00 与更新日期之间的毫秒数

# 转换返回实例方法

方法 描述
toDateString() 以美式英语和人类易读的形式返回一个日期对象日期部分的字符串
toISOString() YYY-MM-DDTHH:mm:ss.sssZ 返回格式字符串
toJSON() YYY-MM-DDTHH:mm:ss.sssZ 返回格式字符串
toLocaleDateString() 返回格式字符串以系统设置地区关联
toLocaleString() 返回日期对象的字符串，该字符串与系统设置的地区关联
toLocaleTimeString() 返回该日期对象时间部分的字符串，该字符串格式与系统设置的地区关联
toString() 返回该日期对象的字符串
toTimeString() 以人类易读形式返回一个日期对象时间部分的字符串，该字符串以美式英语格式化。
toUTCString() 把一个日期对象转换为一个以 UTC 时区计时的字符串
valueOf() 从 1970 年 1 月 1 日 0 时 0 分 0 秒到该日期的毫秒数
