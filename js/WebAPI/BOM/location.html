# location 对象

提供了当前窗口中加载文档的信息，表示其链接到的对象的位置

# 属性

属 性 描述
href 完整URL
protocol 页面使用的协议。通常是"http:"或"https:",最后有一个":"
host 服务器名及端口号
hostname 服务器名或url域名
port 端口
pathname URL 中的路径和（或）文件名,开头有一个“/"
search URL 的查询字符串。这个字符串以问号开头,开头有一个“?”
hash URL 散列值（井号后跟零或多个字符），开头有一个“#”。如果没有则为空字符串
username 域名前指定的用户名
password 域名前指定的密码
origin(只读) URL 的源地址

# 方法

# assign()

触发窗口加载并显示指定的URL的内容

location.assign(url);
参数 描述
url 跳转到的链接

# reload()

刷新当前页面

location.reload(forcedReload);
参数 描述
forcedReload 值为 true 时，将强制浏览器从服务器加载页面资源，
值为 false 或者未传参时，浏览器则可能从缓存中读取页面。

# replace()

URL来替换当前的资源

当前页面不会保存到会话历史中，这样，用户点击回退按钮时，将不会再跳转到该页面。

location.replace(url);
参数 描述
url 新链接地址

# toString()

返回完整的url地址
