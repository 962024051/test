# Windows对象

# 属性

# 窗口

属性 描述
screenX 返回浏览器左边框到左边屏幕边缘的 CSS 像素数
screenLeft 返回浏览器左边框到左边屏幕边缘的 CSS 像素数
screenY 返回浏览器上边框到顶端屏幕边缘的 CSS 像素数
screenTop 返回浏览器上边框到顶端屏幕边缘的 CSS 像素数

# 窗口大小

属性 描述
innerWidth 获得浏览器窗口的内容区域的宽度，包含垂直滚动条(如果有的话)
innerHeight 获得浏览器窗口的内容区域的高度，包含水平滚动条(如果有的话)
outerWidth 获取浏览器窗口外部的宽度,包括侧边栏、窗口镶边和调正窗口大小的边框
outerHeight 获取整个浏览器窗口的高度,包括侧边栏、窗口镶边和调正窗口大小的边框

# 视口

属性 描述
pageXOffset 返回文档/页面水平方向滚动的像素值
scrollX 返回文档/页面水平方向滚动的像素值
screenY 返回文档在垂直方向已滚动的像素值。
screenTop 返回文档在垂直方向已滚动的像素值。

# 导航与打开窗口

属性 描述
closed 窗口是否关闭，true: 窗口已被关闭，false: 窗口是打开的。
opener 返回打开当前窗口的那个窗口的引用

# 方法

# 窗口

当前窗口移动到指定的坐标位置

moveTo()

window.moveTo(x, y)
参数 描述
x 横坐标
y 纵坐标

moveBy()

根据指定的值，移动当前窗口

window.moveBy(x, y)
参数 描述
x 水平方向移动的像素值
y 垂直方向移动的像素值

# 窗口大小

resizeBy()

当前窗口调整到一定的大小。

window.resizeBy(xDelta, yDelta)
参数 描述
x 窗口水平方向变化的像素值
y 窗口垂直方向变化的像素值

resizeTo()

动态调整窗口的大小。（包括滚动条、窗口边框等）

window.resizeTo(aWidth, aHeight)
参数 描述
aWidth 宽度，表示新的outerWidth属性
aHeight 高度，表示新的outerHeight属性

# 视口

scroll()

滚动窗口至文档中的特定位置。

window.scroll(x-coord, y-coord) 
window.scroll(options)
参数 描述
x-coord 置于左上角的像素点的横坐标
y-coord 置于左上角的像素点的纵坐标
options 一个 ScrollToOptions 字典

scrollBy()

在窗口中按指定的偏移量滚动文档

window.scrollBy(x-coord, y-coord);
window.scrollBy(options)
参数 描述
x-coord 水平滚动的偏移量，单位：像素
y-coord 垂直滚动的偏移量，单位：像素
options 一个 ScrollToOptions 字典

scrollTo()

滚动到文档中的某个坐标

window.scrollTo(x-coord,y-coord )
window.scrollTo(options)
参数 描述
x-coord 文档中的横轴坐标
y-coord 文档中的纵轴坐标
options 一个 ScrollToOptions 字典

ScrollToOptions 字典

ScrollToOptions 字典
// 正常滚动
window.scrollTo({
 left: 100,
 top: 100,
 behavior: 'auto'
});
// 平滑滚动
window.scrollTo({
 left: 100,
 top: 100,
 behavior: 'smooth'
});
属性 描述
top Y 轴方向滚动的像素数
left X 轴方向滚动的像素数
behavior smooth(平滑滚动)
instant(瞬间滚动)
默认auto,实测效果等同于instant

# 导航与打开窗口

# open()

打开一个新窗口。

let windowObjectReference =window.open(strUrl, strWindowName, [strWindowFeatures]);



let strWindowFeatures = `
    menubar=yes,
    location=yes,
    resizable=yes,
    scrollbars=yes,
    status=yes
`;
    window.open(
        "http://www.cnn.com/", 
        "CNN_WindowName", 
        strWindowFeatures
    );
参数 描述
windowObjectReference 成功返回新窗口属性和方法，失败返回null
strUrl 新窗口的url地址
strWindowName 新窗口的名称
strWindowFeatures(可选) 传递新窗口配置属性和方法

# close()

关闭当前窗口或某个指定的窗口

window.close();

# 定时器

setInterval()

重复调用一个函数或执行一个代码段，在每次调用之间具有固定的时间延迟。

var intervalID = scope.setInterval(func, delay, [arg1, arg2, ...]);
var intervalID = scope.setInterval(code, delay);
参数 描述
func 重复调用函数
code 函数名字
delay 延迟毫秒数
arg1, ..., argN(可选) 传递func函数的附加参数

setTimeout()

设置一个定时器，该定时器在定时器到期后执行一个函数或指定的一段代码。

var timeoutID = scope.setTimeout(func[, delay, arg1, arg2, ...]);
var timeoutID = scope.setTimeout(func[, delay]); 
var timeoutID = scope.setTimeout(code[, delay]);
参数 描述
func 重复调用函数
code 函数名字
delay 延迟毫秒数
arg1, ..., argN(可选) 传递func函数的附加参数

clearInterval()

取消先前通过 setInterval() 设置的重复定时任务。

scope.clearInterval(intervalID)
参数 描述
intervalID 要取消的定时器的 ID。是由 setInterval() 返回的
返回值 返回undefined

clearTimeout

取消了先前通过调用setTimeout()建立的定时器。

scope.clearTimeout(timeoutID)
参数 描述
intervalID 要取消定时器的标识符。 该ID由相应的setTimeout()调用返回
返回值 返回undefined
