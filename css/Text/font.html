# 字体属性

# 01 字体设置

font-family: [ <family-name> | <generic-family> ];
属性值 描述
<family-name> 字体族的名字
<generic-name> 通用字体族名字

# 02 字体高级印刷

font-feature-settings：normal | <feature-tag-value>
属性值 描述
normal 文本使用默认设置进行布局
<feature-tag-value> 标记值（0,1）的列表被传递到文本布局引擎以启用或禁用字体特征

# 03 字体大小

font-size: <absolute-size> | <relative-size> | <length-percentage>|inherit where
  <absolute-size> = xx-small | x-small | small | medium | large | x-large |
  xx-large <relative-size> = larger | smaller <length-percentage> = <length> |
  <percentage> inherit;
属性值 案例 描述
<absolute-size> xx-small或x-large 绝对大小值
<relative-size> larger 相对大小值
<length> 10px 或 1em 长度值
<percentage> 80% 百分比值
inherit inherit 从父元素继承字体大小

# 04 字体小写字母大小

font-size-adjust: none | <number>;
属性值 案例 描述
none xx-small 或 x-large 根据font-size属性决定字体大小
<number> 1 乘以font-size结果的值定义字体大小

# 05 字体拉伸

font-stretch: <font-stretch-absolute> where <font-stretch-absolute> = normal |
  ultra-condensed | extra-condensed | condensed | semi-condensed | semi-expanded
  | expanded | extra-expanded | ultra-expanded | <percentage>;
属性值 百分比
ultra-condensed 50％
extra-condensed 62.5％
condensed 75％
semi-condensed 87.5％
normal 100％
semi-expanded 112.5％
expanded 125％
extra-expanded 150％
ultra-expanded 200％
<percentage> 自定义百分比值

# 06 字体样式

font-style ： normal | italic | oblique <angle>
属性值 描述
normal 常规字体
italic 斜体
oblique <angle> 倾斜体 (倾斜度数)

# 07 字体变体大写

font-variant-caps: normal | small-caps | all-small-caps | petite-caps |
  all-petite-caps | unicase | titling-caps;
属性值 描述
normal 关闭一切特殊字符变体
small-caps 小型大写字母（OpenType 特性：smcp）
all-small-caps 大小写字母全部转化为小型大写字母（OpenType 特性: c2sc, smcp）
petite-caps 允许特小型大写字母的使用（OpenType 特性: pcap）
all-petite-caps 将大小写字母全部转化为小型大写字母。（OpenType 特性: c2pc, pcap）
unicase 允许将大写字母转化为小型大写字母与普通小写字母的混用 （OpenType 特性: unic）
titling-caps 允许首字母大写（OpenType 特性: titl）

font-variant 相关属性在百度或在线手册看下,里面功能在大都数内容在开发或内侧阶段以及少部分游览器才可以用,这里只讲常用及各大游览器已兼容相关属性

控制备用字体使用

font-variant-alternates: normal | [ stylistic( <feature-value-name> ) || historical-forms || styleset( <feature-value-name># ) || character-variant( <feature-value-name># ) || swash( <feature-value-name> ) || ornaments( <feature-value-name> ) || annotation( <feature-value-name> ) ]
where 
<feature-value-name> = <custom-ident>
属性值 描述
normal 关键字禁用备用字体
historical-forms 关键字启用历史类型-过去常见但今天不常见的字体
stylistic() 可以为个别字体启用字体样式替换
styleset() 函数可以对字符集启用字体样式替换
character-variant() 启用字符的特定样式替代
swash() 启用斜字体
ornaments() 启用装饰物
annotation() 支持注释

# 08 字体粗细

font-weight ：<font-weight-absolute> | bolder | lighter
where
<font-weight-absolute> = normal | bold | <number <a href="/zh-CN/docs/CSS/Value_definition_syntax#Brackets" title="Brackets: enclose several entities, combinators, and multipliers to transform them as a single component">[1,1000]>
属性值 数值 描述
Thin (Hairline) 100 微细
Extra Light (Ultra Light) 200 超细
Light 300 轻细
Normal 400 正常粗细
Medium 500 中粗
Semi Bold (Demi Bold) 600 半粗
Bold 700 加粗
Extra Bold (Ultra Bold) 800 超粗
Black (Heavy) 900 重粗
<number> 定义数值 1-1000 粗细值

# 09 字体行高

line-height：normal | <number> | <length> | <percentage>
属性值 描述
normal 默认值
<number> 数字
<length> 长度
<percentage> 百分比

# 10 简写属性

font: [ [ < "font-style" > || <font-variant-css21> || < "font-weight" > || <
  "font-stretch" > ]? < "font-size" > [ / < "line-height" > ]? < "font-family" >
  ] | caption | icon | menu | message-box | small-caption | status-bar where
  <font-variant-css21> = [ normal | small-caps ];

font:[样式 或 字母大小写 或 粗细 或 拉伸]? 字号/行高？ 字体设置
where
样式+字母大小写+粗细+拉伸+字号/行高+字体设置
