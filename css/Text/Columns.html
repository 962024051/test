# 多列布局

# 01 元素列数

column-count ：<integer> | auto
属性值 描述
<integer> 设置元素的列数量
auto 列的数量自动计算并由其他 CSS 属性指定, 例如 column-width

# 02 元素列宽

column-width ：<length> | auto
属性值 描述
<length> 设置元素列的宽度（长度）
auto 列的宽度自动计算并由其他 CSS 属性指定, 例如 column-count

# 03 元素列数量和列宽简写

columns ：<column-width> || <column-count>

/* 列宽 */
columns: 18em;

/* 列数 */
columns: auto;
columns: 2;

/*数量+宽度 */
columns: 2 auto;
columns: auto 12em;
columns: auto auto;
属性值 描述
<column-width> 元素的列宽度
<column-count> 元素的列数量

# 04 列填充

column-fill ： auto | balance | balance-all
属性值 描述
auto 列按顺序自动填充，但部分列有空白区域
balance 内容在各列平均分配，最后一列是平衡
balance-all 内容在各列平均分配，所有列是平衡

# 05 多列间距

column-gap ： normal | <length> | <percentage>
属性值 描述
normal 默认间距 1em
<length> 长度值 间距
<percentage> 百分比 间距

# 06 元素跨列

column-span : none | all
属性值 描述
normal 元素不跨多个列
all 元素横跨所有列

# 07 多列之间线条规则

# 线条颜色

column-rule-color : <color>
属性值 描述
<color> 颜色

# 线条样式

column-rule-style : <'border-style'>
属性值 描述
<border-style> 参考边框线条样式

# 线条宽度

column-rule-width ： <length> | thin | medium | thick
属性值 描述
<length> 长度值设置边线宽度
thin 细边线
medium 中等边线
thick 宽边线

# 简写属性

column-rule : <width> || <style> || <color>
属性值 描述
<width> 线条宽度
<style> 线条样式
<color> 线条颜色
