# 内容

在元素的 ::before 和 ::after 伪元素中插入内容

content : normal | none | [ <content-replacement> | <content-list> ] [/ <string> ]?
where 
<content-replacement> = <image>
<content-list> = [ <string> | contents | <image> | <quote> | <target> | <leader()> ]+
属性值 描述
none 不会产生伪类元素
normal :before 和 :after 伪类元素中会被视为 none
<string> 字符数据类型
<uri> 链接数据类型
<counter> 计数器数据类型
attr(X) 元素的X属性以字符串形式返回
open-quote**|**close-quote 这些值会被 quotes 中定义的字符串替换
no-open-quote**|**no-close-quote 不会生产任何内容，但是会改变（增加或降低）引号层级

# 引号的样式

quotes ： none | auto | [ <string> <string> ]+

/* 关键字 */
quotes: none;

/* 字符 */
quotes: "«" "»";           
quotes: "«" "»" "‹" "›";
属性值 描述
none content 属性的值 open-quote 和 close-quote 将不会展示引号
auto 适当的引号，基于在所选元素上设置的任何语言值
<string> 字符数据类型 一组或者多组 <string> 的值对应 open-quote and close-quote.
第一对表示引号的外层，第二对表示第一个嵌套层，下一对表示第三层，依此类推。

# 计数器增加值

counter-increment:[ <custom-ident> <integer>? ]+ | none

/* 将此计算器增加1  */
counter-increment: my-counter;

/* 将此计算器增加 5 */
counter-increment: my-counter 5;

/* 将“ counter1”增加1，并将“ counter2”减少4*/
counter-increment: counter1 counter2 -4;
属性值 描述
<custom-ident> 自定义标识数据类型
<integer> 整型数据类型 默认值1 增加或减少数量
none 不得增加计数器。用作默认值，或取消更具体规则的增量

# 计数器重置值

counter-reset:[ <custom-ident> <integer>? ]+ | none

/* 将此计算器重置1  */
counter-reset: my-counter;

/* 将此计算器重置 5 */
counter-reset: my-counter 5;

/* 将“ counter1”重置1，并将“ counter2”重置4*/
counter-reset: counter1 counter2 -4;
属性值 描述
<custom-ident> 自定义标识数据类型
<integer> 整型数据类型 默认值1 重置数量
none 关键字表示不执行计数器重置

# 计数器设置值

counter-set:[ <custom-ident> <integer>? ]+ | none

/* 将此计算器重置1  */
counter-set: my-counter;

/* 将此计算器重置 5 */
counter-set: my-counter 5;

/* 将“ counter1”重置1，并将“ counter2”重置4*/
counter-set: counter1 counter2 -4;
属性值 描述
<custom-ident> 自定义标识数据类型
<integer> 整型数据类型 默认值1 设置数量
none 无需设置计数器

# 计数器

返回一个代表计数器的当前值的字符串

counters( <custom-ident>, <counter-style>? )

where
<counter-style> = <counter-style-name> | symbols()

where
<counter-style-name> = <custom-ident>

/* 简单使用 */
counters(计数器名称);

/* 更改计数器显示 */
counters(countername, upper-roman)
属性值 描述
<custom-ident> 自定义标识数据类型
<counter-style> 计数器样式

# 计数器规则

@counter-style <counter-style-name> {
    system: <counter system>
    symbols: <counter symbols>
    additive-symbols: <additive-symbols>
    negative: <negative symbol>
    prefix: <prefix>
    suffix: <suffix>
    range: <range>
    pad: <padding>
    speak-as: <speak-as>
    fallback: <counter-style-name>
}
属性 描述
system 指定一个算法，用于将计数器的整数值转化为字符串表示
negative 指定一个符号，当计数器表示的值为负的时候，把这个符号加在值的前面或后面
prefix 指定一个符号，加在标记表示符的前面。
前缀在最后阶段才会被加上，所以在计数器的最终表示中，它在negative前。
suffix 与prefix类似，suffix指定一个符号，加在标记表示符的后面。
range 指定一个counter-style生效的范围，如果计数器的值不再这个范围内，那么自定义的counter-style不会生效，counter-style会后退到fallback的style
pad 在你想要给标记表示符最小值时使用。
fallback 定义一个备用的系统，当自定义的系统不能使用或者计数器的值超过了定义的范围时使用。
symbols 定义一个符号，用于标记的表示。符号可以包含字符串，图片或自定义的识别码
additive-symbols 尽管symbols属性中指定的符号可以被system中定义的大部分算法所使用
speak-as 定义如何在语音识别器中读出计数器样式，比如屏幕阅读器。例如基于该描述符的值，标记符号的值可以作为有序列表的数字或者字幕作为无序列表的音频提示读出。

详情请阅官方文档
