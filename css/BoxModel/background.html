# 背景属性

# 01 滚动

background-attachment ： scroll | fixed | local
属性值 描述
fixed 背景固定
scroll 背景随着窗体滚动而滚动
local 背景随着元素滚动而滚动

# 02 裁剪

background-clip ：  border-box | padding-box | content-box | text
属性值 描述
border-box 背景在元素边框区域裁剪
padding-box 背景在元素内边距区域裁剪
content-box 背景在元素内容区域裁剪
text 背景在元素文字区域裁剪

# 03 颜色

background-color ： <color>
属性值 描述
<color> <color>颜色

# 04 图片

background-image: none | <image>;
属性值 描述
none 无背景图
<image> <image>图片格式

# 05 参考原点位置摆放

background-origin ：  border-box | padding-box | content-box | text
属性值 描述
border-box 背景摆放元素边框区域
padding-box 背景摆放元素内边距区域
content-box 背景摆放元素内容区域

# 06 设置图像位置

# 1 水平位置

background-position-x : [ center | [ [ left | right | x-start | x-end ]? <length-percentage>? ]! ]
where
<length-percentage> = <length> | <percentage>

/* 值 */
background-position-x: left;
background-position-x: center;
background-position-x: right;

/* 百分比 */
background-position-x: 25%;

/* 长度值 */
background-position-x: 0px;
background-position-x: 1cm;
background-position-x: 8em;

/* 边缘值 */
background-position-x: right 3px;
background-position-x: left 25%;

/* 多值 */
background-position-x: 0px, center;
属性值 描述
left 设置在左边位置
center 设置在中间位置
right 设置在右边位置
<length> 自定义长度值相对边缘边缘，边缘默认在左边
<percentage> 自定义百分比相对边缘边缘，边缘默认在左边

# 2 垂直位置

background-position-y :[ center | [ [ top | bottom | y-start | y-end ]? <length-percentage>? ]! ]
where
<length-percentage> = <length> | <percentage>

/* 值 */
background-position-y: top;
background-position-y: center;
background-position-y: bottom;

/* 百分比 */
background-position-y: 25%;

/*  长度值 */
background-position-y: 0px;
background-position-y: 1cm;
background-position-y: 8em;

/* 边缘值  */
background-position-y: bottom 3px;
background-position-y: bottom 10%;

/* 多值 */
background-position-y: 0px, center;
属性值 描述
top 设置在上边位置
center 设置在中间位置
bottom 设置在下边位置
<length> 自定义长度值相对边缘边缘，边缘默认在上边
<percentage> 自定义百分比相对边缘边缘，边缘默认在上边

# 3 简写属性

background-position :<bg-position>#
where
<bg-position> = [ [ left | center | right | top | bottom | <length-percentage> ] | [ left | center | right | <length-percentage> ] [ top | center | bottom | <length-percentage> ] | [ center | [ left | right ] <length-percentage>? ] && [ center | [ top | bottom ] <length-percentage>? ] ]
where
<length-percentage> = <length> | <percentage>

/* 值*/
background-position: top;
background-position: bottom;
background-position: left;
background-position: right;
background-position: center;

/* 百分比x值和y值 */
background-position: 25% 75%;

/* 长度x值和y值 */
background-position: 0 0;
background-position: 1cm 2cm;
background-position: 10ch 8em;

/* 多值*/
background-position: 0 0, center;

/* x边缘值和y边缘值 */
background-position: bottom 10px right 20px;
background-position: right 3em bottom 10px;
background-position: bottom 10px right;
background-position: top right 10px;
属性值 描述
<length> 长度值
<percentage> 百分比值
left 设置在左边位置
right 设置在右边位置
top 设置在上边位置
bottom 设置在下边位置
center 设置在中间位置

# 07 图像平铺

background-repeat ：<repeat-style>#
where
repeat-x | repeat-y | [ repeat | space | round | no-repeat ]{1,2}

/* 单值语法 */
background-repeat: repeat-x;
background-repeat: repeat-y;
background-repeat: repeat;
background-repeat: space;
background-repeat: round;
background-repeat: no-repeat;

/* 双值语法: 水平horizontal | 垂直vertical */
background-repeat: repeat space;
background-repeat: repeat repeat;
background-repeat: round space;
background-repeat: no-repeat round;
单值 等价于双值 描述
repeat-x repeat no-repeat 水平平铺
repeat-y no-repeat repeat 垂直平铺
repeat repeat repeat 全部平铺
space space space 背景图像自动缩放直到适应且填充满整个容器
round round round 背景图像相同间距平铺且填充满整个容器或某个方向
no-repeat no-repeat no-repeat 不平铺

# 08 图像尺寸大小

background-size : <bg-size>#
where
<bg-size> = [ <length-percentage> | auto ]{1,2} | cover | contain
where
<length-percentage> = <length> | <percentage>

/* 关键字 */
background-size: cover
background-size: contain

/* 一个值: 这个值指定图片的宽度，图片的高度隐式的为auto */
background-size: 50%
background-size: 3em
background-size: 12px
background-size: auto

/* 两个值 */
/* 第一个值指定图片的宽度，第二个值指定图片的高度 */
background-size: 50% auto
background-size: 3em 25%
background-size: auto 6px
background-size: auto auto

/* 逗号分隔的多个值：设置多重背景 */
background-size: auto, auto     /* 不同于background-size: auto auto */
background-size: 50%, 25%, 25%
background-size: 6px, auto, contain
属性值 描述
<length> 长度值
<percentage> 百分比值
auto 自动计算相等比例大小背景图像
cover 背景完全覆盖，可能会有背景溢出
contain 图片不溢出的放在容器中，可能会漏出部分区域

# 09 简写属性

background: [ <bg-layer>, ] * <final-bg-layer> where <bg-layer> = <bg-image> ||
    <bg-position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box> ||
    <box> <final-bg-layer> = <background-color> || <bg-image> || <bg-position>
    [ / <bg-size> ]? || <repeat-style> || <attachment> || <box> || <box>
    各个属性可以任意顺序摆放;
属性值 完整属性值 描述
<background-color> background-color 背景图像
<bg-image> background-image 图片
<bg-position> background-position 图像位置
<bg-size> background-size 图像尺寸大小
<repeat-style> background-repeat 图像平铺
<attachment> background-attachment 图像滚动
<box> background-clip 和 background-origin 图像裁剪和原点位置
