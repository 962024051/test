# 用户基本布局

# 盒子能见度

visibility ： visible | hidden | collapse
属性值 描述
visible 元素正常显示
hidden 隐藏元素，但是其他元素的布局不改变，相当于此元素变成透明。要注意若将其子元素设为 visibility: visible，则该子元素依然可
collapse 主要用来隐藏表格的行或列。隐藏的行或列能够被其他内容使用。对于表格外的其他对象，其作用等同于 hidden。

# 盒子大小

box-sizing ： content-box | border-box
属性值 区域效果 描述
content-box width = 内容的宽度
height = 内容的高度		 默认值，标准盒子模型。 width 与[height 只包括内容的宽和高， 不包括边框（border），内边距（padding），外边距（margin）
border-box width = border + padding + 内容的宽度
height = border + padding + 内容的高度		 width和 height属性包括内容，内边距和边框，但不包括外边距

# 盒子阴影

/* 插页(阴影向内)  x偏移量 | y偏移量 | 阴影模糊半径 | 阴影扩散半径 | 阴影颜色 */
box-shadow ： inset? [<offset-x>  <offset-y> || <blur-radius>  || <spread-radius> ]  && <color>

box-shadow ： inset? && <length>{2,4} && <color>?
/* x偏移量 | y偏移量 | 阴影颜色 */
box-shadow: 60px -16px teal;

/* x偏移量 | y偏移量 | 阴影模糊半径 | 阴影颜色 */
box-shadow: 10px 5px 5px black;

/* x偏移量 | y偏移量 | 阴影模糊半径 | 阴影扩散半径 | 阴影颜色 */
box-shadow: 2px 2px 2px 1px rgba(0, 0, 0, 0.2);

/* 插页(阴影向内) | x偏移量 | y偏移量 | 阴影颜色 */
box-shadow: inset 5em 1em gold;

/* 任意数量的阴影，以逗号分隔 */
box-shadow: 3px 3px red, -1em 0 0.4em olive;
属性值 描述
inset 如果没有指定inset，默认阴影在边框外，即阴影向外扩散。
使用 inset 关键字会使得阴影落在盒子内部，这样看起来就像是内容被压低了。
此时阴影会在边框之内 (即使是透明边框）、背景之上、内容之下
<offset-x> 水平方向偏移
<offset-y> 垂直方向偏移
<blur-radius> 阴影模糊半径，值越大，模糊面积越大，阴影就越大越淡。 不能为负值
<spread-radius> 阴影扩散半径，取正值时，阴影扩大；取负值时，阴影收缩。默认为 0
<color> 颜色

# 鼠标指针光标

cursor：[ [ <url> [ <x> <y> ]? , ]* [ auto | default | none | context-menu | help | pointer | progress | wait | cell | crosshair | text | vertical-text | alias | copy | move | no-drop | not-allowed | e-resize | n-resize | ne-resize | nw-resize | s-resize | se-resize | sw-resize | w-resize | ew-resize | ns-resize | nesw-resize | nwse-resize | col-resize | row-resize | all-scroll | zoom-in | zoom-out | grab | grabbing ] ]
属性值 描述
<url> url 地址 数据类型
<x>  <y> 可选 x，y 坐标。两个小于 32 的无单位非负数
关键字 鼠标系统自带的关键字指针光标 可以参考 css 文档查阅

# 鼠标指针光标颜色

caret-color: auto | <color>
属性值 描述
auto 默认颜色
<color> 颜色 数据类型

# 轮廓

设置一个元素的轮廓

# 轮廓厚度

outline-width： <length> | thin | medium | thick
属性值 描述
<length> 长度值 数据类型
thin 取决于用户代理。通常等同于桌面浏览器的 1px
medium 取决于用户代理。通常等同于桌面浏览器的 3px
thick 取决于用户代理。通常等同于桌面浏览器的 5px

# 轮廓样式

outline-style ： auto | <border-style>
属性值 描述
<border-style> 边框样式 参考边框样式类型
auto 自动显示轮廓样式
none 没有轮廓样式

# 轮廓颜色

outline-color ：<color> | invert
属性值 描述
<color> 颜色数据类型
invert 反色,用于确认轮廓的显示.注意不是所有浏览器都支持该属性,若不则该属性无效.

# 轮廓间隙

一个元素边缘或边框之间的间隙

outline-offset ：<length>
属性值 描述
<length> 长度值数据类型

# 简写属性

outline： [ <'outline-color'> || <'outline-style'> || <'outline-width'> ]

/* 样式 */
outline: solid;

/* 颜色 | 样式 */
outline: #f66 dashed;

/* 样式 | 宽度 */
outline: inset thick;

/* 颜色 | 样式 | 宽度 */
outline: green solid 3px;
属性值 描述
<outline-color> 颜色
<outline-style> 样式
<outline-width> 宽度

# 元素调整大小

允许你控制一个元素的可调整大小性

resize ： none | both | horizontal | vertical | block | inline
属性值 描述
none 元素不能被用户缩放
both 允许用户在水平和垂直方向上调整元素的大小
horizontal 允许用户在水平方向上调整元素的大小。
vertical 允许用户在垂直方向上调整元素的大小
block 允许用户在块方向上调整元素的大小
inline 允许用户在行内方向上调整元素的大小

# 选中文本

user-select: auto | text | none | contain | all;
属性值 描述
none 元素及其子元素的文本不可选中
auto 元素及其子元素的文本自动选中
text 用户可以选择文本。
contain 允许在元素内选择；但是，选区将被限制在该元素的边界之内
all 双击元素选中文本
