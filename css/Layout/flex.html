# 弹性布局

# 基础单位(父盒子设置)

# 初始化盒子空间

flex-basis ： content | <'width'>
属性值 描述
content 基于 flex 的元素的内容自动调整大小空间
<width> 参考宽度 数据类型

# 扩展剩余空间比率

剩余盒子空间进行按份数分配各自盒子扩展空间

flex-grow ： <number>

当前盒子初始化大小=外边距+边框+内边距+内容空间
当前盒子扩展空间=剩余容器空间 / 总份数 * 当前盒子设置份数 + 当前盒子空间
属性值 描述
<number> 扩展值，不能负值

# 收缩溢出空间比率

溢出盒子空间进行按份数分配各自盒子缩小空间

flex-shrink  ： <number>

总空间集合=盒子1*缩小1比例+盒子2*缩小2比例+盒子3*缩小3比例+。。。盒子n*缩小n比例
当前收缩盒子空间= (当前盒子初始大小 * 缩小比例) / 总空间集合 * 溢出空间
当前盒子最终空间=当前盒子初始大小 - 当前收缩盒子空间
属性值 描述
<number> 缩放值，不能负值

# 简写属性

flex：none|[<'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]

/* 基础值 */
flex: auto;
flex: initial;
flex: none;
flex: 2;

/*一个值，数字类型: flex-grow 扩展剩余空间  */
flex: 2;

/*一个值，长度数据类型（宽度/高度）: flex-basis 初始化盒子空间*/
flex: 10em;
flex: 30px;

/* 俩个值: flex-grow | flex-basis */
/* 俩个值: 扩展剩余空间 | 初始化盒子空间 */
flex: 1 30px;

/* 俩个值: flex-grow | flex-shrink */
/* 俩个值: 扩展剩余空间 | 伸缩溢出空间 */
flex: 2 2;

/* 三个值: flex-grow | flex-shrink | flex-basis  */
/* 俩个值: 扩展剩余空间 | 伸缩溢出空间  | 初始化盒子空间 */
flex: 2 2 10%;
属性值 描述
auto 自动放大扩展或缩小溢出盒子空间 ，相当于 flex: 1 1 auto
none 不会扩展或缩小盒子空间，相当于 flex: 0 0 auto
<flex-grow> 按比例进行扩展剩余空间
<flex-shrink> 按比例进行缩小溢出空间
<flex-basis> 初始化盒子空间大小

# 方向与拆行(子盒子设置)

# 方向

控制盒子元素排列的方向。

 flex-direction ： row | row-reverse | column | column-reverse
属性值 描述
row 元素盒子沿着 X 轴（水平方向）依次从左到右排列
row-reverse 元素盒子沿着 X 轴（水平方向）依次从右到左排列
column 元素盒子沿着 Y 轴（垂直方向）依次从顶部到底部排列
<flex-shrink> 元素盒子沿着 Y 轴（垂直方向）依次从底部到顶部排列

# 拆行

flex-wrap: nowrap | wrap | wrap-reverse;
属性值 描述
nowrap 盒子单行排列，部分盒子会溢出容器外边
wrap 盒子换行排列，不会溢出容器
wrap-reverse 与 warp 换行方向相反，其余效果一样

# 简写属性

flex-flow:<'flex-direction'> || <'flex-wrap'>

/* 单值: <'flex-direction'> */
flex-flow: row;
flex-flow: row-reverse;
flex-flow: column;
flex-flow: column-reverse;

/* 单值: <'flex-wrap'> */
flex-flow: nowrap;
flex-flow: wrap;
flex-flow: wrap-reverse;

/* 双值: <'flex-direction'> and <'flex-wrap'> */
flex-flow: row nowrap;
flex-flow: column wrap;
flex-flow: column-reverse wrap-reverse;
属性值 描述
<flex-direction> 控制
<flex-wrap> 盒子换行排列，不会溢出容器

# 排序

order ： <integer>
属性值 描述
<integer> 整型类型 控制弹性盒子位置 数字越小越在前面
