# 布局显示类型

# 布局显示类型型介绍

模式 一行排列 设置宽度/高度 默认宽度 包含
块级盒子 一个 可以 内容填满父盒子 任何所有标签
内联/行内盒子 多个 不可以 内容本身大小 文本或行内相关元素
行内块盒子 多个 可以 内容本身大小

# 布局显示类型

# 外部

只会直接影响自己元素的外部表现，而不影响自己儿孙级元素的表现

 <display-outside> ：block | inline | run-in
属性值 描述
block 渲染块级盒子，前后有换行符(独占一行)
inline 渲染内联盒子，前后没有换行符(不独占一行)，和其他盒子在同一行
run-in 根据后面的元素进行转换 display 为 inline | inlin-block | block

# 内部

管束自己的儿子级元素的排布的，规定它们的布局

<display-inside> ： flow | flow-root | table | flex | grid | ruby
属性值 描述
flow 渲染某种类型盒子与外部元素的显示（display）类型盒子有关。（实验开发中）
flow-root 渲染块级盒子，建立一个新的块格式上下文，可以用来清除浮动，去除margin合并，实现两栏自适应布局等
table 类似 table 表格 html 标签
flex 弹性布局
grid 网络布局
ruby 类似汉字上面标注汉语拼音效果

# 列表

实现列表元素的效果，等同于<ul><li>

<display-listitem> ： list-item
属性值 描述
list-item 实现列表元素的效果，等同于<ul><li>

# 内部复杂

<display-internal> ：table-* | ruby-*
属性值 描述
table-header-group 类似<thead>于 HTML 元素
table-row-group 类似<tbody>于 HTML 元素
table-footer-group 类似<tfoot>于 HTML 元素
table-row 类似<tr>于 HTML 元素
table-cell 类似<td>于 HTML 元素
table-column-group 类似<colgroup>于 HTML 元素
table-column 类似<col>于 HTML 元素
table-caption 类似<caption>于 HTML 元素
ruby-base 类似<rb>于 HTML 元素
ruby-text 类似<rt>于 HTML 元素
ruby-base-container 类似<rbc>于 HTML 元素
ruby-text-container 类似<rtc>于 HTML 元素

# 是否有该类型

 <display-box> ： contents | none
属性值 描述
none 关闭盒子显示，但不影响布局
contents 元素本身不渲染盒子，但子元素和伪元素渲染盒子

# 内联

<display-legacy> ： inline-block | inline-table | inline-flex |inline-grid
属性值 描述
inline-block 内联块级
inline-table 内联表格
inline-flex 内联弹性布局
inline-grid 内联网络布局

# 简写属性

display ： [ <display-outside> || <display-inside> ] | <display-listitem> | <display-internal> | <display-box> | <display-legacy>
where
<display-outside> = block | inline | run-in
<display-inside> = flow | flow-root | table | flex | grid | ruby
<display-listitem> = <display-outside>? && [ flow | flow-root ]? && list-item
<display-internal> = table-row-group | table-header-group | table-footer-group | table-row | table-cell | table-column-group | table-column | table-caption | ruby-base | ruby-text | ruby-base-container | ruby-text-container
<display-box> = contents | none
<display-legacy> = inline-block | inline-list-item | inline-table | inline-flex | inline-grid
属性值 描述
<display-outside> 外部盒模型
<display-inside> 内部盒模型
<display-listitem> 列表盒模型
<display-internal> 复杂内部盒模型
<display-box> 是否有盒模型
<display-legacy> 内联盒模型
