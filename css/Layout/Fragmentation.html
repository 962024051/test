# 分片化

# 盒装饰断开

定义当元素跨多行、多列或多页时，元素的片段应如何呈现。

box-decoration-break: slice | clone
属性值 描述
slice 元素被按照盒子被切割前的原始样式渲染之后，针对每个行/列/页面将此假设框渲染成片段
clone 每个框片段与指定的边框、填充和边距独立呈现

# 容器之前断开

页面，列或区域在生成的盒子之前应如何处理断开

break-before ： auto | avoid | always | all | avoid-page | page | left | right | recto | verso | avoid-column | column | avoid-region | region
属性值 描述
auto 主体框之前不强制断开
avoid 主体框之前避免断开
always 主体框之前强制断开，要在上下文类型
all 主体框之前强制断开
avoid-page 主体框之前避免出现任何分页符
page 主体框之前强制分页符
left 主体框之前插入分页符，以使下一页成为左页为准
right 主体框之前插入分页符，以使下一页成为右页为准
recto 主体框之前插入分页符，以使下一页成为记录页为准
verso 主体框之前插入分页符，以使下一页成为反页符为准
avoid-column 主体框之前避免出现任何分列
column 主体框之前强制换行。
avoid-region 主体框之前避免出现任何区域中断
region 主体框之前强制区域中断
