# 函数

# attr()

获取选择到的元素的某一HTML属性值

 attr( attribute-name <type-or-unit>? [, <fallback> ]? )
属性值 描述
attribute-name 是CSS所引用的HTML属性名称。
<type-or-unit> 表示所引用的属性值的单位。
如果该单位相对于所引用的属性值不合法，那么attr()表达式也不合法
<allback> 如果HTML元素缺少所规定的属性值或属性值不合法

# calc()

在声明 CSS 属性值时执行一些计算

calc( <calc-sum> )where<calc-sum>=<calc-product>[['+'|'-']<calc-product>]*where<calc-product>=<calc-value>['*'<calc-value>|'/'<number>]*

where 
<calc-value> = <number> | <dimension> | <percentage> | ( <calc-sum> )

/* property: calc(expression) */
width: calc(100% - 80px);

<length>、<frequency>, <angle>、<time>、<percentage>、<number>、或 <integer>
关键词 描述
+ 加法
- 减法
/ 乘法，乘数中至少有一个是 <'number'>
* 除法，除数（/ 右面的数）必须是 <'number'>

# clamp()

作用是把一个值限制在一个上限和下限之间，当这个值超过最小值和最大值的范围时，在最小值和最大值之间选择一个值使用。它接收三个参数：最小值、首选值、最大值。

clamp( <calc-sum>#{3} )
where 
<calc-sum> = <calc-product> [ [ '+' | '-' ] <calc-product> ]*

where 
<calc-product> = <calc-value> [ '*' <calc-value> | '/' <number> ]*

where 
<calc-value> = <number> | <dimension> | <percentage> | ( <calc-sum> )

clamp() 被用在 <length>、<frequency>、<angle>、<time>、<percentage>、<number>、<integer> 中都是被允许的

clamp(MIN, VAL, MAX) 其实就是表示 max(MIN, min(VAL, MAX))

clamp(1rem, 2.5vw, 2rem); 
clamp(1.5rem, 2.5vw, 4rem);

# env()

CSS 函数以类似于 var 函数和 custom properties 的方式将用户代理定义的环境变量值插入你的 CSS 中。区别在于，环境变量除了由用户代理定义而不是由用户定义外，还被全局作用在文档中，而自定义属性则限定在声明它们的元素中。 为了告诉浏览器使用屏幕上所有的可用空间，

env( <custom-ident> , <declaration-value>? )
属性值 描述
<custom-ident> 字符串数据类型

# fit-content()

根据其中的公式将指定尺寸限制为有效尺寸

fit-content([ <length> | <percentage> ] )

fit-content(<length-percentage>) == min(max-content, max(min-content,<length-percentage>))
fit-content(200px) == min(max-content, max(min-content, 200px))
属性值 描述
<length> 长度值数据类型
<percentage> 百分比数据类型

# max()

接受一个或多个用逗号分隔的表达式作为他的参数，数值最大的表达式的值将会作为指定的属性的值。

max( <calc-sum># )
where 
<calc-sum> = <calc-product> [ [ '+' | '-' ] <calc-product> ]*
where 
<calc-product> = <calc-value> [ '*' <calc-value> | '/' <number> ]*
where 
<calc-value> = <number> | <dimension> | <percentage> | ( <calc-sum> )

width: max(10vw, 4em, 80px);

# mix()

一个或多个逗号分隔符表达式作为参数，表达式的值中最小的值作为参数值。

min( <calc-sum># )
where 
<calc-sum> = <calc-product> [ [ '+' | '-' ] <calc-product> ]*
where 
<calc-product> = <calc-value> [ '*' <calc-value> | '/' <number> ]*
where 
<calc-value> = <number> | <dimension> | <percentage> | ( <calc-sum> )

width: min(50vw, 200px);

# image-set()

浏览器从一组给定选择最适当的CSS图象，主要用于高像素密度屏幕的方法

image-set() = image-set( <image-set-option># ) 
where <image-set-option> = [ <image> | <string> ] <resolution> and
      <string> is an <url>
 
background-image: image-set( "cat.png" 1x,
                             "cat-2x.png" 2x,
                             "cat-print.png" 600dpi);

# repeat()

重复副本

repeat( [ <integer> | auto-fill | auto-fit ] , <track-list> )

repeat(重复次数,设置重复属性值)

repeat(4, 300px) == 300px 300px 300px 300px
属性值 描述
<integer> 重复次数是整型数据类型 最小值 1
auto-fill 自动按比例重复次数以最小值(宽度或高度)为基准单位 填充剩余空间会有空白
auto-fit 自动按比例重复次数以最小值(宽度或高度)为基准单位 填充剩余空间会分配各个重复属性身上
<track-list> 设置重复列表属性集合，里面可以设置长度，百分比，网格份数等数据类型相关属性值

# minmax()

自定义区间范围

minmax( [ <length> | <percentage> | min-content | max-content | auto ] , [ <length> | <percentage> | <flex> | min-content | max-content | auto ] )

minmax(最大值,最小值)
属性值 描述
<length> 长度值数据类型
<percentage> 百分比数据类型
<flex> 网络布局数据类型
max-content 网格的轨道长度自适应内容最大的那个单元格
min-content 网格的轨道长度自适应内容最小的那个单元格
auto 最大值格按 max-content,最小值按min-width/min-height

# var()

调用自定义属性

var( <custom-property-name> , <declaration-value>? )background-color: var(
  --main-bg-color
);
关键词 描述
<custom-property-name> 自定义属性名
<declaration-value> 声明值（后备值）,<'custom-property-name'>调用变量没有声明，就使用该参数调用

# --*

自定义属性 (--*)：CSS 变量

--* : <declaration-value>

--somekeyword: left;
--somecolor: #0000ff;
--somecomplexvalue: 3px 6px rgb(20, 32, 54);
属性值 描述
<declaration-value> 由一个或者多个语法执行出来，只要这些语法是正确合理的，不包含非法语句。这个值就理应是有效语法执行出来的值。
