# 属性

# 背景滤镜

为一个元素后面区域添加图形效果（如模糊或颜色偏移）

backdrop-filter ：none | <filter-function-list>
where 
<filter-function-list> = [ <filter-function> | <url> ]+

/* 关键词值 */
backdrop-filter: none;

/* 指向 SVG 滤镜的 URL */
backdrop-filter: url(commonfilters.svg#filter);

/* <filter-function> 滤镜函数值 */
backdrop-filter: blur(2px);
backdrop-filter: brightness(60%);
backdrop-filter: contrast(40%);
backdrop-filter: drop-shadow(4px 4px 10px blue);
backdrop-filter: grayscale(30%);
backdrop-filter: hue-rotate(120deg);
backdrop-filter: invert(70%);
backdrop-filter: opacity(20%);
backdrop-filter: sepia(90%);
backdrop-filter: saturate(80%);

/* 多重滤镜 */
backdrop-filter: url(filters.svg#filter) blur(4px) saturate(150%);
属性 描述
none 没有应用于背景的滤镜
<filter-function-list> 一个以空格分隔的滤镜函数（<filter-function>）或是要应用到背景上的 SVG 滤镜

# 滤镜

模糊或颜色偏移等图形效果应用于元素。滤镜通常用于调整图像，背景和边框的渲染。

filter ：none | <filter-function-list>
where 
<filter-function-list> = [ <filter-function> | <url> ]+

/* 关键词值 */
filter: none;

/* 指向 SVG 滤镜的 URL */
filter: url(commonfilters.svg#filter);

/* <filter-function> 滤镜函数值 */
filter: blur(2px);
filter: brightness(60%);
filter: contrast(40%);
filter: drop-shadow(4px 4px 10px blue);
filter: grayscale(30%);
filter: hue-rotate(120deg);
filter: invert(70%);
filter: opacity(20%);
filter: sepia(90%);
filter: saturate(80%);

/* 多重滤镜 */
filter: url(filters.svg#filter) blur(4px) saturate(150%);
属性 描述
none 没有应用于背景的滤镜
<filter-function-list> 一个以空格分隔的滤镜函数（<filter-function>）或是要应用到背景上的 SVG 滤镜

# 数据类型

<filter-function>

# 高斯模糊图像

blur(radius)
属性 描述
radius 模糊的半径，值为<length>长度数据类型

# 图像亮度

让图像更明亮或更暗淡

brightness(amount)
brightness([<number> | <percentage>])
属性 描述
<number> 数字类型 默认值为 1，高于 1 为亮，低于 1 为暗
<percentage> 百分比类型 默认值为 100%，高于 100%为亮，低于 100%为暗

# 图像对比度

contrast(amount)
contrast([<number> | <percentage>])
属性 描述
<number> 数字类型 默认值为 1， 高于 1 为增加对比度，低于 1 为减少对比度
<percentage> 百分比类型 默认值为 100% ， 高于 100%为增加对比度，低于 100%为减少对比度

# 图像阴影

drop-shadow(offset-x offset-y blur-radius spread-radius color)
属性 描述
offset-x 水平距离，负值左侧，正值右侧
offset-y 垂直距离，负值上方，正值下方
blur-radius 阴影的模糊半径，<length>数据类型。值越大，阴影就越大，也越模糊
spread-radius 阴影的扩展半径，<length>数据类型。正值阴影扩大和变大，负值阴影缩小
color 阴影的颜色

# 图像灰度

grayscale([<number> | <percentage>])
属性 描述
<number> 数字类型 0-1之间的灰度，0为原图像
<percentage> 百分比类型 0-100%之间的灰度，0%为原图像

# 图像色调旋转

hue-rotate(angle)
属性 描述
<angle> 角度旋转数据类型 色调旋转

# 图像颜色反转

invert([<number> | <percentage>])
属性 描述
<number> 数字类型 0-1之间，1为完全反转颜色
<percentage> 百分比类型 0-100%之间，100%为完全反转颜色

# 图像透明

opacity([<number> | <percentage>])
属性 描述
<number> 数字类型 0-1之间，0为完全透明
<percentage> 百分比类型 0-100%之间，0为完全透明

# 图像饱和度

saturate([<number> | <percentage>])
属性 描述
<number> 数字类型 0-1之间，0为完全不包含
<percentage> 百分比类型 0-100%之间，0为完全不包含

# 图像转为棕褐色

sepia([<number> | <percentage>])
属性 描述
<number> 数字类型 0-1之间，1为完全棕褐色
<percentage> 百分比类型 0-100%之间，100%0为完全棕褐色
