# 属性

# 形状图像阈值

shape-image-threshold: <number> | <percentage>;
属性值 描述
<number> 0-1 之间透明度 数字数据类型
<percentage> 0-100% 之间透明度 百分比数据类型

# 形状外边距

shape-margin: <number> | <percentage>;
属性值 描述
<number> 形状外边距 数字数据类型
<percentage> 形状外边距 百分比数据类型

# 外形

shape-outside : none | <shape-box> || <basic-shape> | <image>
属性值 描述
none 该浮动区域不产生影响，行内元素以默认的方式包裹着该元素的margin box。
<shape-box> 根据浮动元素的边缘形状计算出浮动的区域
margin-box 定义一个由外边距的外边缘封闭形成的形状
border-box 定义一个由边界的外边缘封闭形成的形状
padding-box 定义一个由内边距的外边缘封闭形成的形状
content-box 定义一个由内容区域的外边缘封闭形成的形状
<basic-shape> 基于<basic-shape>数据类型计算浮动的区域
<image> 提取并且计算指定 image图像的alpha通道得出浮动区域

# 数据类型

<basic-shape>

# 长方形

inset( <shape-arg>{1,4} [round <border-radius>]? )
属性值 描述
<shape-arg> 长方形与相关盒模型的上，右，下与左边界和顶点的偏移量
<border-radius> 参考border-radius边框圆角属性

# 圆形

circle( [<shape-radius>]? [at <position>]? )
属性值 描述
<shape-radius> 圆的半径
<position> 百分比数据类型

# 椭圆形

ellipse( [<shape-radius>{2}]? [at <position>]? )
属性值 描述
<shape-radius> 代表了 rx 与 ry，其中 rx 代表了 x 轴方向的半径， ry 代表了 y 轴方向的半径
<position> 百分比数据类型

# 多边形

polygon( [<fill-rule>,]? [<shape-arg> <shape-arg>]# )
属性值 描述
<fill-rule> 多边形规则
<shape-arg> 偏移量

# SVG 路径

path( [<fill-rule>,]? <string>)
属性值 描述
<fill-rule> 多边形规则
<string> 字符串
