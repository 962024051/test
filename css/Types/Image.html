# 渐变

# 线性渐变

  /*
  角度 或 to 上下 左右,颜色 起始位置 结束位置 ，颜色 起始位置 结束位置  ，颜色 起始位置 结束位置  */
linear-gradient(
  [ <angle> | to <side-or-corner> ,]? <color-stop-list> )
  \---------------------------------/ \----------------------------/
        渐变线声明定义                           颜色停止列表

where <side-or-corner> = [ left | right ] || [ top | bottom ]
  and <color-stop-list> = [ <linear-color-stop> [, <color-hint>? ]? ]#, <linear-color-stop>
  and <linear-color-stop> = <color> [ <color-stop-length> ]?
  and <color-stop-length> = [ <percentage> | <length> ]{1,2}
  and <color-hint> = [ <percentage> | <length> ]
linear-gradient([ [ [ <angle> | to [top | bottom] || [left | right] ],]? <color-stop>[, <color-stop>]+);

/* 角度 ， 蓝色 ， 红色*/
linear-gradient(45deg, blue, red);

/* 位置 ，蓝色， 红色 */
linear-gradient(to left top, blue, red);

/* 角度，蓝色，绿色 当前结束位置 ，红色 */
linear-gradient(0deg, blue, green 40%, red);
属性值 示例 描述
<angle> 45deg 用角度值指定渐变的方向（或角度）。角度顺时针增加
<side-or-corner> to top left to 水平位置(left right) 垂直位置（top bottom）
<linear-color-stop> green 40% 渐变颜色并且有当前起始位置或结束位置
百分比或长度值数据类型 起始结束位置

# 径向渐变


/* 形状 || 圆形半径 at 圆心位置 ， 颜色结束列表 */
radial-gradient() = radial-gradient(
  [ <ending-shape> || <size> ]? [ at <position> ]? ,
  <color-stop-list>
)

radial-gradient() = radial-gradient(
/* 圆形 半径  at 圆心位置 */
  [ [ circle               || <length> ]                          [ at <position> ]? , |
  /* 椭圆形 水平圆形半径 垂直圆形半径  at 圆心位置 */
    [ ellipse              || <length-percentage>{2} ]            [ at <position> ]? , |
    /* 圆形状 半径关键字 at 圆心位置 */
    [ [ circle | ellipse ] || <extent-keyword> ]                  [ at <position> ]? , |
    at <position> ,
  ]?
  /*  颜色结束列表 */
  <color-stop> [ , <color-stop> ]+
)
<extent-keyword> = closest-corner | closest-side | farthest-corner | farthest-side
属性值 描述
ellipse 椭圆形
circle 圆形
<length> 长度值 圆形半径
<length-percentage>{2} 长度值或百分比 圆形水平半径和圆形垂直半径
<position> 圆形位置
一个值是 关键字上下左右，长度，百分比等数据类型设置位置等
两个值 水平位置，垂直位置 他们数据类型是长度，百分比
<color-stop> 颜色，开始位置和结束位置
closest-corner 渐变的边缘形状与容器距离渐变中心点最近的一边相切（圆形）
或者至少与距离渐变中心点最近的垂直和水平边相切（椭圆）。
closest-side 渐变的边缘形状与容器距离渐变中心点最近的一个角相交。
farthest-corner 与 closest-side 相反，边缘形状与容器距离渐变中心点最远的一边相切（或最远的垂直和水平边）。
farthest-side 渐变的边缘形状与容器距离渐变中心点最远的一个角相交。

# 修正图像取向

image-orientation: from-image | none;
属性值 描述
none 没有任何图像旋转
from-image 适当旋转

# 图像渲染

image-rendering ： auto | crisp-edges | pixelated
属性值 描述
auto 默认值，应使用最大化图像外观的算法来缩放图像
crisp-edges 保留图像中的对比度和边缘的算法来缩放图像，并且不会在处理中使图像变得光滑或模糊。
pixelated 向上缩放图像时，必须使用“最近邻居”或类似的算法，以便图像看起来由大像素组成。缩小时，这与“自动”相同

# 控制内容框对象大小

object-fit ： fill | contain | cover | none | scale-down
属性值 描述
fill 默认，不保证保持原有的比例，内容拉伸填充整个内容容器
contain 保持原有尺寸比例。内容被缩放。
cover 保持原有尺寸比例。但部分内容可能被剪切
none 被替换的内容将保持其原有的尺寸
scale-down 内容的尺寸与 none 或 contain 中的一个相同，取决于它们两个之间谁得到的对象尺寸会更小一些。

object-position：<position>where<position>=[[left|center|right]||[top|center|bottom]|[left|center|right|<length-percentage>][top|center|bottom|<length-percentage>]? | [ [ left | right ] <length-percentage> ] && [ [ top | bottom ] <length-percentage> ] ]
where
<length-percentage> = <length> | <percentage>

/* 单值 */
object-position: 50%;

/* 双值 */
/* 水平方向  垂直方向 */

object-position: right top;
属性值 描述
<position> 水平方向 垂直方向 设置位置
<length> 长度值数据类型 位置
<percentage> 百分比数据类型 位置
